# Statistical Analysis of Aerial Phenomena

A project based in python that focuses on the analysis and exploration of an aerial phenomina dataset.

## Overview

This project investigates patterns and trends within an aieral phenomena (specifically UAP) sightings data set trhoguh:

- Data cleaning and preperation
- Exploratory data analysis (EDA)
- Data Visualization
- Pattern detection
- Temporal and geographical analysis

The goal of this project is to better understand trends in reporting, types of anomalies, and correlations in the reported aerial phenomena data.

---

## Features

- Cleaning and preprocessing of a data set
- Visualizing time-based trends
- Analysis of geographical distribution
- Detection of anomalies and outliers
- Creating correlations throught scatter plots

## Instalation

Clone the repository:

```bash
git clone https://github.com/FischerLogan/statistical_analysis_of_aerial_phenomena.git
cd statistical_analysis_of_aerial_phenomena
```

Install dependencies:

```bash
pip install -r requirements.txt
```

---

## Usage

Run the Jupyter notebooks:

```bash
jupyter notebook
```

Or run Python scripts directly:

```bash
python main.py
```
